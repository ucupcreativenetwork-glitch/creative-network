# Creative Network — Company Profile Website

Website company profile modern untuk **Creative Network**, studio kreatif digital berbasis di Yogyakarta, Indonesia.

## ✨ Fitur

- Single-page application yang cepat & elegan
- Desain modern (dark theme + indigo accent)
- Fully responsive (mobile-first)
- Tailwind CSS via CDN (tanpa build step)
- Interaktif:
  - Mobile hamburger menu
  - Portfolio filterable (Branding / Web / Video / Marketing)
  - Modal detail proyek
  - Animated counters
  - Smooth scroll + active nav
  - Contact form (simulasi submit + toast)
- Optimasi SEO dasar + meta tags
- Bahasa Indonesia

## 🚀 Cara Menjalankan

Cukup buka file `index.html` di browser:

```bash
# Buka langsung
start index.html          # Windows
open index.html           # macOS
```

Atau gunakan live server sederhana:

```bash
# Jika ada Python
python -m http.server 8000

# Atau PHP
php -S localhost:8000
```

## 📁 Struktur Proyek

```
creative-network/
├── index.html      # Website utama (semua dalam satu file)
└── README.md
```

## 🛠 Teknologi

- HTML5 + Tailwind CSS (Play CDN)
- Vanilla JavaScript
- Font Awesome 6
- Responsive design

## 📝 Catatan Pengembangan

Website ini dibuat sebagai starting point profesional. 
Silakan ganti:

- Konten teks sesuai real company profile
- Foto/gambar dengan aset asli
- Warna brand jika berbeda
- Link sosial media & kontak
- Harga layanan (jika ingin ditampilkan)

Untuk production lebih advanced, bisa di-migrasi ke:
- Vite + Tailwind
- Astro
- Next.js / Nuxt
- Atau tetap single file untuk kemudahan hosting GitHub Pages

## Deploy ke GitHub Pages

1. Push branch `main` (sudah otomatis)
2. Settings → Pages → Source: Deploy from a branch → `main` / root
3. Website akan tersedia di `https://ucupcreativenetwork-glitch.github.io/creative-network`
